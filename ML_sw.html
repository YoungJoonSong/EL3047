머신러닝 알고리즘 구현과 실험을 위하여 Python 프로그램 언어과 관련된 다음과 같은 패키지를 많이 사용할 예정입이다.

  • numpy: Array processing for numbers, strings, records, and objects
  • scipy: Scientific Library for Python
  • scikit-learn : A set of python modules for machine learning and data mining

Anaconda는 위의 패키지를 포함하여 여기에 설명된 많은 패키지를 포함하고 있어서 아나콘다를 설치하는 것도 좋은 방법입니다. 그리고 아나콘다에 텐서플로를 설치 하면 좀 더 쉽게 머신러닝과 딥러닝을 배울 수 있게 된다. Anaconda Python tensorflow installation and environment set up을 참조하여 아나콘다 환경에서 텐서플로우의 세부 기능을 추가하여 사용한다. 텐서플로우의 기초 는 텐스플로우를 이해하는데 도움이 된다.
